# Knapsack Algorithm

Java implementation of the 0/1 Knapsack problem using dynamic programming.

## What it does

Given a set of items with weights and values, and a maximum capacity,
finds the optimal combination of items that maximizes total value
without exceeding the weight limit.

## Tech Stack

- Java
- Dynamic Programming

## How to run

1. Clone the repository
```bash
   git clone https://github.com/yaren-atici/knapsack-algorithm.git
```
2. Open in IntelliJ IDEA or any Java IDE
3. Run `Main.java`
